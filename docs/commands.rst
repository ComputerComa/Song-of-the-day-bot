===================
Command Permissions
===================
The Following commands require `Administrator` or `Manage Server` permissions
* ``/announce```
* ``/get-announcement-history``
* ``/get_suggestion_history``
* ``/mark_suggestion_used``
* ``/pick_random_suggestion``
